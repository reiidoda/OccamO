# Security Policy

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub Security Advisory
for this repository. Do not disclose vulnerabilities in public issues or pull
requests.

## Supported Versions

Only the latest release series is supported for security fixes. If you are
using an older version, upgrade to the latest release before reporting.

## Response Expectations

Reports are triaged on a best-effort basis. You will receive acknowledgement
and status updates as the issue is investigated.
